# Week 2: Coffee Shop (3 Interacting Classes)

**Topic:** Objects, classes, and instances (Hillar Chapter 1 & 2).

**Goal:** Mini-project with **3 interacting classes** and a **simple UI** (console).

## Classes
- `MenuItem` holds item details.
- `Order` stores selections and computes totals.
- `CoffeeShop` manages menu lookup and order creation.

## Run
```bash
python main.py
```
